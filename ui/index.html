Tenant Management

Tenants are managed via Kubernetes CRD. Create tenants with:

kubectl apply -f - <<EOF
apiVersion: genesis.ai/v1
kind: Tenant
metadata:
  name: acme-bank
spec:
  displayName: "ACME Bank AG"
  quota: enterprise
  isolation: network
  oidcGroup: acme-admins
  esgEnabled: true
  complianceFrameworks:
    - eIDAS
    - GDPR
    - PSD2
    - Basel-III
EOF
